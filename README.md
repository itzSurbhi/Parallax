# 🌄 Parallax Scrolling Website

A simple and visually appealing **Parallax Scrolling Website** built with **HTML** and **CSS**.  
This project demonstrates how to use perspective, transform, and background-attachment to create smooth scrolling effects with layered images.

---

## 🚀 Features
- 🎨 Modern **parallax scrolling effect**
- 🖼️ Background and foreground layers with depth
- 📜 Smooth section transitions with descriptive content
- 🌍 Responsive design basics with `viewport` support
- ✨ Minimal and clean styling

---

## 📂 Project Structure
parallax-project/
│── index.html # Main HTML file
│── parallax.css # CSS for parallax effects & styling
│── mount.avif # Background image (mountains)
│── foreground image.png # Foreground image
│── biking.webp # Section background (biking)
│── paragliding.webp# Section background (paragliding)
│── surfing.webp # Section background (surfing).

---

## 🛠️ Technologies Used
- **HTML5**
- **CSS3 (Flexbox, Transform, Perspective, Parallax)**
